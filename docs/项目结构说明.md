## 1. 总体结构（传统 Web 工程）

本项目当前采用传统的 Java Web 工程目录结构（非 Maven）：

- `src/`：Java 源码与类路径资源
- `src/config/`：Spring/MyBatis/日志/数据库配置
- `WebContent/`：Web 根目录（JSP、静态资源、`WEB-INF`）

## 2. 分层结构（SSM 三层）

按 SSM 的典型分层理解：

- 表现层（Controller）：`src/com/ischoolbar/programmer/controller/**`
  - 前台：`.../controller/home/**`
  - 后台：`.../controller/admin/**` 与 `.../controller/system/**`
- 业务层（Service）：`src/com/ischoolbar/programmer/service/**`
  - 负责业务编排、事务边界（由 Spring 管理）
- 数据访问层（DAO/Mapper）：`src/com/ischoolbar/programmer/dao/**`
  - 通过 MyBatis 映射文件执行 SQL
- 实体（Entity）：`src/com/ischoolbar/programmer/entity/**`
  - 与数据库表字段对应的数据对象

## 3. 配置文件位置

- Web 配置：`WebContent/WEB-INF/web.xml`
  - Spring 容器配置：`classpath:config/spring/applicationContext.xml`
  - Spring MVC 配置：`classpath:config/springmvc/springmvc.xml`
- Spring：`src/config/spring/applicationContext.xml`
  - 包扫描：`com.ischoolbar.programmer`
  - 事务管理：`DataSourceTransactionManager` + `@Transactional`
  - MyBatis：`SqlSessionFactoryBean` + `MapperScannerConfigurer`
- Spring MVC：`src/config/springmvc/springmvc.xml`
  - Controller 扫描：`com.ischoolbar.programmer.controller`
  - 视图解析：`/WEB-INF/views/ + *.jsp`
  - 拦截器：后台/前台登录拦截（`interceptor/**`）
- MyBatis：`src/config/mybatis/`
  - 核心配置：`mybatis-config.xml`
  - Mapper XML：`mapper/**`、`mapper/admin/**`
- 数据库连接：`src/config/db.properties`
- 日志：`src/config/log4j.properties`

## 4. 视图与静态资源

- JSP 视图：`WebContent/WEB-INF/views/`
  - 后台模块页面：`views/user`、`views/role`、`views/menu`、`views/book_order` 等
  - 前台页面：`views/home/**`
- 静态资源：`WebContent/resources/`
  - 后台资源：`resources/admin/**`
  - 前台资源：`resources/home/**`
